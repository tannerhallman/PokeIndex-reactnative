# OffTheGrid-reactnative
This project is going to blow your mind.

##Helpful Commands
If you are having trouble, start with these commands:
<br />

```
rm -rf ./node_modules
watchman watch-del-all
npm cache clean
npm install
react-native run-ios/run-android
```
